# Training Dashboard

A simple training dashboard with quick access to training links.

## Features
- Join Meeting button
- April 20th Class tile with Teams meeting link
- Mark Attendance
- Learning Portal

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to your repository on GitHub: https://github.com/Ad4rshS/training.com
2. Click on "Settings" tab
3. Scroll down to "Pages" section in the left sidebar
4. Under "Build and deployment", select "Deploy from a branch"
5. Under "Branch", select "master" and "/ (root)"
6. Click "Save"
7. Your site will be available at: https://ad4rsh.github.io/training.com/

## Local Development

To run this locally:
```bash
python -m http.server 8000
```
Then open http://localhost:8000 in your browser.
